## **Получение связанных ресурсов по заявке** 

**Получение списка заявителей** 

_Вызов_ 

GET 

= = = = = /api/taskcreator?serviceid {serviceid}&fields {fieldList}&seach {searchString}&pagesize {value}&page 

{value} 

Получить пользователей сервиса, которые могут быть заявителями на заявках этого сервиса. Возвращает коллекцию объектов ресурса Пользователь (User). 

_Возможные параметры_ 

_**serviceid** – обязательный. Идентификатор сервиса._ 

_**fieldList** – список полей для получения (поля ресурса Пользователь)_ 

_**search** – строка поиска. Ищет в полях «Имя пользователя» и «Логин»._ 

_**pagesize** ,_ _**page** - параметры отображения списка_ 

**Получение списка исполнителей**
