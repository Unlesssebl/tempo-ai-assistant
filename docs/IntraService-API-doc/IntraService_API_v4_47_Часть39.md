## Фильтрация списка статусов 

_Поля фильтрации_ 

|**Поле фильтрации**|**Тип передаваемого**<br>**значения**|**Описание фильтрации**|
|---|---|---|
|ChangedLessThan|DateTime|Дата последнего изменения<br>статусов меньше или равна<br>указаннойдате|
|ChangedMoreThan|DateTime|Дата последнего изменения<br>статусов больше или равна<br>указаннойдате|
|Changed|DateTime|Дата последнего изменения<br>статусов равна указанному дню<br>(время не учитывается, т.е. с<br>00:00до 23:59указанногодня)|
|IsInitial|bool|Статусы, у которых проставлен<br>признак «Начальный»|
|IsFixed|bool|Статусы, у которых проставлен<br>признак «Заявка выполнена»|
|IsFinal|bool|Статусы, у которых проставлен<br>признак «Конечный»|
|IsCommentRequired|bool|Статусы, у которых проставлен<br>признак «Комментарии<br>обязательны»|
|IsConcurrence|bool|Статусы, у которых проставлен<br>признак «Согласование»|
|IsExternal|bool|Статусы, у которых проставлен<br>признак «Во внешней<br>организации»|
|IsDeadlineAccountingPaused|bool|Статусы, у которых проставлен<br>признак «Учет времени<br>выполнения приостановлен»|



_Примеры запросов_ 

- 1) Получим все статусы для согласования: _GET_ /api/taskstatus?IsConcurrence=true 

- 2) Получим все статусы с приостановленным временем выполнения и измененные 25 октября 2015 года. 

_GET_ /api/taskstatus?IsDeadlineAccountingPaused=true&Changed=2015-10-25
