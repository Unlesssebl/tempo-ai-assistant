## Инструкция по первому запуску

Совет по развертыванию на Ubuntu:
После того как файлы окажутся на сервере, вместо копирования .venv, выполните:

bash
# Установить uv, если его нет
curl -LsSf https://astral.sh/uv/install.sh | sh
source $HOME/.cargo/env
# Синхронизировать зависимости (создаст чистый .venv под Linux)
uv sync
# Обновить базу данных векторов
uv run scripts/update_database.py
# Запустить бота
uv run run_bot.py

### 1. Подготовка окружения (Windows)
Убедитесь, что у вас установлен Python 3.12+.
```powershell
# Создание виртуального окружения
python -m venv .venv
# Активация
.venv\Scripts\activate
# Установка зависимостей (рекомендуется использовать uv для скорости)
pip install uv
uv sync
```

### 2. Настройка конфигурации
1. Скопируйте файл `env.example` и переименуйте его в `.env`.
2. Откройте `.env` и заполните обязательные поля:
   * `TELEGRAM_TOKEN`: Токен вашего бота от @BotFather.
   * `GEMINI_API_KEY`: Ваш API ключ Google AI Studio.
   * (Опционально) Настройки IntraService, если нужна работа с заявками.

### 3. Индексация документов
Поместите ваши документы (PDF, DOCX, XLSX) в папку `data/` и запустите процесс обработки:
```powershell
python scripts/process_documents.py
```

### 4. Запуск компонентов
Вы можете запускать компоненты по отдельности (в разных окнах терминала):

*   **Сервер (API):**
    ```powershell
    python run_server.py
    ```
*   **Telegram-бот:**
    ```powershell
    python run_bot.py
    ```
*   **Десктопный клиент:**
    ```powershell
    python run_client.py
    ```
