# Программа частичной компенсации расходов по уплате процентов по ипотечным кредитам

## Условия для участия

1. **Стаж работы** должен составлять более 6 месяцев
2. **Отсутствие фактов нарушения трудовой дисциплины**
3. Сотрудник должен являться **заемщиком по ипотечному кредитованию**

---

## Необходимые документы

| Документ |
|----------|
| Заявка установленной формы |
| Справка-заявка установленной формы за текущий год об отсутствии фактов нарушения трудовой дисциплины |
| Копия договора займа (кредита) |
| Копия договора участия в долевом строительстве |
| Копия договора купли-продажи |
| Копия выписок банка или квитанций, подтверждающих уплату процентов работником лично |
| Справка об остатке задолженности по уплате процентов по ипотечному кредиту |
