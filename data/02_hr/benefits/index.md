# Льготы и компенсации

Группа компаний ТЭМПО предоставляет ряд дополнительных льгот и программ поддержки для своих сотрудников.

### Доступные программы
- **[[Компенсация ипотеки|mortgage_compensation]]**: Условия получения компенсации процентов по ипотечным кредитам.
- **[[Приведи друга|refer_a_friend]]**: Реферальная программа по подбору персонала с денежным вознаграждением.
